# React Shop Catalog — домашнее задание 12

Учебное React-приложение по теме **«Основной и продвинутый React. Компоненты, состояние и хуки»**.

## Что реализовано

- маршрутизация через `react-router-dom`;
- страницы `/`, `/list`, `/list/:id`, `/about`;
- дополнительная страница `/favorites` для избранных товаров;
- навигационное меню через `NavLink` с активным состоянием ссылок;
- загрузка списка товаров из API;
- загрузка детальной информации о товаре по `id`;
- состояние загрузки;
- обработка ошибок API;
- глобальное состояние через Context API;
- добавление и удаление товаров из избранного;
- сохранение избранного в `localStorage`.

## API

По умолчанию используется публичный API:

```env
VITE_API_URL=https://dummyjson.com
```

Если нужно подключить свой backend из предыдущего модуля, измените `VITE_API_URL` в `.env` и при необходимости адаптируйте файл:

```text
src/api/productsApi.js
```

## Структура проекта

```text
src/
  api/
    productsApi.js          # функции для запросов к API
  components/
    ErrorMessage.jsx        # сообщение об ошибке
    Layout.jsx              # общий макет и меню
    Loader.jsx              # индикатор загрузки
    ProductCard.jsx         # карточка товара
  context/
    FavoritesContext.jsx    # избранные товары
    ProductsContext.jsx     # список товаров и кеширование данных
  pages/
    About.jsx               # страница о проекте
    Details.jsx             # подробная страница товара
    Favorites.jsx           # избранные товары
    Home.jsx                # главная страница
    List.jsx                # список товаров
    NotFound.jsx            # страница 404
  App.jsx                   # маршруты приложения
  main.jsx                  # точка входа
  styles.css                # стили
```

## Как запустить проект

1. Установить зависимости:

```bash
npm install
```

2. Запустить проект:

```bash
npm run dev
```

3. Открыть приложение в браузере:

```text
http://localhost:5173
```

## Как проверить

Проверьте маршруты:

- `/` — главная страница;
- `/list` — список товаров из API;
- `/list/1` — подробная информация о товаре;
- `/about` — информация о проекте;
- `/favorites` — список избранных товаров.

Проверьте функциональность:

1. На странице `/list` должны отображаться карточки товаров.
2. При нажатии на кнопку **Подробнее** открывается страница товара.
3. На странице товара отображаются название, цена, рейтинг, категория, описание и изображение.
4. Кнопка **Добавить в избранное** добавляет товар в глобальное состояние.
5. В меню изменяется счётчик избранных товаров.
6. На странице `/favorites` отображаются добавленные товары.
7. Если API недоступно, приложение показывает сообщение об ошибке.

## Деплой

Проект можно развернуть на Vercel или Netlify.

Для Vercel:

1. Загрузить проект в открытый GitHub-репозиторий.
2. Зайти на Vercel.
3. Нажать **Add New Project**.
4. Выбрать репозиторий.
5. Framework Preset: **Vite**.
6. Build Command: `npm run build`.
7. Output Directory: `dist`.
8. После деплоя добавить ссылку на приложение в ЛМС.
