# React Shop — домашние задания 12 и 13

Проект сделан как пример интернет-магазина на React. В нём реализованы маршрутизация, загрузка данных из API, глобальное состояние, избранное, оптимизация компонентов, тестирование и подготовка к развёртыванию.

## Используемые технологии

- React
- React Router
- Context API
- JavaScript
- Vite
- Jest
- React Testing Library
- localStorage
- Lighthouse

## Реализованный функционал

### Домашнее задание 12

- Добавлена маршрутизация через React Router.
- Реализованы страницы:
  - `/` — главная страница;
  - `/list` — каталог товаров;
  - `/list/:id` — подробная страница товара;
  - `/about` — информация о проекте.
- Добавлено меню с `NavLink` и активным состоянием ссылок.
- Список товаров загружается через API.
- На странице каталога отображаются карточки товаров.
- На странице товара отображаются название, описание, метаинформация и изображение.
- Добавлены состояния загрузки и ошибки.
- Для хранения данных используется Context API.
- Реализовано добавление товара в избранное.

### Домашнее задание 13

- Добавлена страница `/favourites` для избранных товаров.
- В избранном отображаются:
  - название товара;
  - количество;
  - категория;
  - цена;
  - итоговая сумма по товару.
- Реализовано удаление товара из избранного.
- Реализована очистка всего списка избранного.
- Избранное сохраняется в `localStorage` и не пропадает после перезагрузки страницы.
- Добавлена оптимизация:
  - `React.memo` для компонентов;
  - `useMemo` для вычисляемых данных;
  - `useCallback` для функций, которые передаются в дочерние компоненты;
  - `React.lazy` и `Suspense` для ленивой загрузки страниц.
- Добавлены тесты на:
  - отображение страниц;
  - проверку маршрутов;
  - загрузку каталога;
  - добавление и удаление товаров из избранного.

## Установка и запуск

Перед запуском нужно установить Node.js LTS.

Проверка установки:

```bash
node -v
npm -v
```

Установка зависимостей:

```bash
npm install
```

Запуск проекта в режиме разработки:

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5173/
```

## Запуск тестов

```bash
npm test
```

Запуск тестов с отчётом о покрытии:

```bash
npm run test:coverage
```

## Production build

```bash
npm run build
```

Проверка production build локально:

```bash
npm run preview
```

## Проверка Lighthouse

После запуска проекта или деплоя нужно открыть приложение в Google Chrome и выполнить проверку:

1. Открыть DevTools.
2. Перейти во вкладку Lighthouse.
3. Выбрать категории Performance и Accessibility.
4. Нажать Analyze page load.

Требование по заданию:

- Performance > 80;
- Accessibility > 90.
